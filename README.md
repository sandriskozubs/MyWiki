# What is MyWiki? :cd:

**MyWiki** is a local Wikipedia, in which you can add your own articles.

You can use it as a place, where you store information thats relevant to yourself.

# How does MyWiki work?

Administrators can **view**, **edit**, **create** and **delete** articles.

Articles are saved with creation dates and edit dates.

It will be possible to add text and images to articles.
**!! Images are not yet implemented !!**

It will also be possible to search articles by keywords.
**!! Not possible to search yet !!**

MyWiki's database tables:

Administrator table

    | id | username | password | role | 

Article table

    | id | title | content | created_at | updated_at |
